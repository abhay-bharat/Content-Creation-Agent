# Production Checklist

Use this checklist before publishing any post. All checkboxes should be ✅ before shipping.

## Pre-Production

### Concept Phase
- [ ] Idea validated against brand pillars?
- [ ] Aligned with audience (software engineers, builders)?
- [ ] Decision filter: *Does this help someone think better about AI?*
- [ ] Format selected (reel/carousel/story/series)?
- [ ] Effort estimate reasonable (1-3 hours?)?

### Hook Phase
- [ ] Hook written and tested?
- [ ] Hook type identified (Surprise/Clarify/Opportunity)?
- [ ] Would this stop *your own* scroll?
- [ ] Hook specific and concrete (not generic)?

### Content Phase
- [ ] Core teaching point is clear (one idea)?
- [ ] Technical accuracy verified (no oversimplification)?
- [ ] Example/proof included (code/diagram/data)?
- [ ] CTA defined (save/share/follow/comment)?
- [ ] Outline follows recommended structure?

## Script & Copy Phase

- [ ] Script written in human voice (not robotic)?
- [ ] Engineer-to-engineer tone (respects audience)?
- [ ] Specific details included (numbers, names, tools)?
- [ ] No marketing fluff or overpromises?
- [ ] Contractions used naturally?
- [ ] Passive voice minimized?
- [ ] Word count fits format (check templates)?

## Visual Phase

- [ ] Consistent with AIThinkingLab style (minimal, dark, contrast)?
- [ ] One core idea per frame/slide?
- [ ] Text large and readable (<1 sec hierarchy)?
- [ ] Color palette consistent (1-2 accent colors)?
- [ ] No clutter, white space used?
- [ ] Motion supports meaning (not decorative)?
- [ ] Brand elements present (logo/watermark)?
- [ ] Font sizes: 3 max, high contrast?

## Format-Specific Checks

### Reel (20-35s)
- [ ] Hook: 3-5 sec opening
- [ ] Pacing: Holds attention throughout
- [ ] Audio: Clear, good levels, no background noise
- [ ] Captions: Synced with audio
- [ ] Frame rate: 1080x1920, 30fps
- [ ] Music: Royalty-free, matches tone
- [ ] CTA visible at end (2-3 sec)

### Carousel (5-10 slides)
- [ ] Slide 1: Hook + preview
- [ ] Slides 2-4: One teaching point per slide
- [ ] Slide 5+: Proof/framework/takeaway
- [ ] Last slide: CTA clear
- [ ] Captions on each slide
- [ ] Consistent style across slides

### AI Signals (5 slides)
- [ ] Slide 1: Hook ("X just happened")
- [ ] Slides 2-3: What happened (facts)
- [ ] Slide 4: Why it matters (connections)
- [ ] Slide 5a: Your analysis (ThinkingLab take)
- [ ] Slide 5b: Source link + credibility

### Story (6-9 frames)
- [ ] Quick load (<3 sec per frame)
- [ ] Clear CTA or engagement hook
- [ ] On-brand colors/fonts
- [ ] Captions readable at phone size

## Quality Bar (Critical)

- [ ] **Useful?** Would I save/reference this in 3 months?
- [ ] **Hook strong?** Does it actually stop scroll?
- [ ] **Visually clean?** Clear hierarchy in <1 sec?
- [ ] **Technically accurate?** No misleading claims?
- [ ] **Sounds human?** Natural voice, not corporate?
- [ ] **Audience fit?** Engineers would get value?
- [ ] **Brand aligned?** Reflects channel personality?

## Analytics Readiness

Before publishing:
- [ ] Optimal time selected (Tue-Thu typically strong)
- [ ] Caption written with strong CTA
- [ ] Hashtags ready (#AIThinkingLab #AIEngineering etc)
- [ ] Cross-post plan (stories, DM tease, etc)?
- [ ] Scheduled in Meta Business Suite

## Post-Publishing (24-48h)

- [ ] Saved analytics screenshot at 24h
- [ ] Replied to all comments
- [ ] Noted what worked (hook, format, topic)
- [ ] Recorded performance in weekly sheet
